# QA Checklist

用于发布前手动验收。建议按顺序勾选，并记录发现的问题。

## 1. 环境与启动

- [ ] `pnpm install` 成功
- [ ] `pnpm dev` 成功启动
- [ ] 首屏 loading 正常显示
- [ ] 进入主界面后 loading 消失
- [ ] 页面无白屏、无明显样式错乱
- [ ] 浏览器控制台无新的红色报错

## 2. 自动化校验

- [ ] `pnpm lint` 通过
- [ ] `pnpm test` 通过
- [ ] `pnpm exec tsc --noEmit` 通过
- [ ] `pnpm build` 通过

## 3. 待机态

- [ ] 初始进入时处于待机状态
- [ ] 未开始时显示“起始行”而不是“消除行”
- [ ] 未开始时显示 `speedStart` 而不是 `speedRun`
- [ ] Logo 待机动画正常显示
- [ ] 未开始时 `←/→` 可调整起始速度
- [ ] 未开始时 `↑/↓` 可调整起始行数
- [ ] 调整后的数值显示正确

## 4. 开始与基础流程

- [ ] `Space` 可以开始游戏
- [ ] `P` 在待机态可开始游戏
- [ ] `R` 在待机态可开始游戏
- [ ] 开始后当前方块正确生成
- [ ] Next 预览正确显示下一块
- [ ] 方块按当前速度自动下落
- [ ] 方块落底后能锁定并生成下一块
- [ ] 游戏过程中无卡死、无明显掉帧

## 5. 键盘操作

- [ ] `←` 左移正常
- [ ] `→` 右移正常
- [ ] `↑` 旋转正常
- [ ] `↓` 软降正常
- [ ] `Space` 硬降正常
- [ ] `P` 暂停正常
- [ ] 再按 `P` 可以恢复
- [ ] `R` 可重开
- [ ] `S` 可切换音效开关
- [ ] 长按 `←/→` 连发节奏正常
- [ ] 长按 `↓` 软降连发节奏正常
- [ ] 撞墙时不会出现异常抖动或穿墙

## 6. 屏幕按钮 / 触控

- [ ] 屏幕上的方向按钮可点击
- [ ] 屏幕上的硬降按钮可点击
- [ ] 屏幕上的暂停按钮可点击
- [ ] 屏幕上的重开按钮可点击
- [ ] 屏幕上的音效按钮可点击
- [ ] 触摸操作不会出现双触发
- [ ] 长按屏幕方向键时连发正常
- [ ] 移动端/模拟触控下布局可用

## 7. 消行 / 计分 / 速度

- [ ] 单行消除正常
- [ ] 双行同时消除正常
- [ ] 三行同时消除正常
- [ ] 四行同时消除正常
- [ ] 消行动画正常播放
- [ ] 消除后棋盘结果正确
- [ ] 每次落地基础分增加正常
- [ ] 消行奖励分增加正常
- [ ] 消除行数统计正确
- [ ] 达到提速条件后速度等级提升正常
- [ ] 最高分记录更新正常

## 8. 暂停 / 焦点切换

- [ ] 暂停后方块停止下落
- [ ] 暂停时不会继续累计状态变化
- [ ] 恢复后方块继续下落
- [ ] 切换标签页/窗口失焦后游戏暂停推进
- [ ] 回到页面后游戏恢复正常
- [ ] 焦点切换后不会突然多掉几格

## 9. 持久化

- [ ] 游戏进行中刷新页面可恢复状态
- [ ] 待机态刷新页面无异常
- [ ] 暂停态刷新页面无异常
- [ ] 接近 game over 时刷新页面无异常
- [ ] 最高分能正确持久化
- [ ] 起始速度与起始行设置能正确恢复或回退

## 10. Game Over / 重开

- [ ] 顶行被占后能触发 game over
- [ ] game over 动画正常播放
- [ ] game over 后回到可重新开始的状态
- [ ] game over 后 `Space` 可重新开始
- [ ] game over 后 `P` 可重新开始
- [ ] game over 后 `R` 可重新开始

## 11. 音效

- [ ] 开始音效正常
- [ ] 左右移动音效正常
- [ ] 旋转音效正常
- [ ] 硬降音效正常
- [ ] 消行音效正常
- [ ] game over 音效正常
- [ ] 关闭音效后所有音效静音
- [ ] 重新开启音效后恢复正常

## 13. 响应式与布局

- [ ] 常规桌面窗口布局正常
- [ ] 缩小浏览器窗口后布局仍可用
- [ ] 高屏窗口下缩放正常
- [ ] 窄屏窗口下不出现严重破版
- [ ] Guide 区域显示正常
- [ ] 主机身、屏幕、侧边状态区未错位

## 14. 发布前最终确认

- [ ] 已完成自动化校验
- [ ] 已完成手动试玩验收
- [ ] 无阻塞级 bug
- [ ] 无明显视觉缺陷
- [ ] 无阻塞级交互问题
- [ ] 可进入发布/提交流程
